Solução

  • Criado: src/components/RoleGuard.tsx — Componente reutilizável com useRef-based state machine

Como funciona

/dashboard/fotografophotographer/dashboard/{role_do_usuario}
/dashboard/clienteclient/dashboard/{role_do_usuario}

Se não logado → redireciona para /login. Se logado sem permissão → redireciona para o dashboard correto da role.
